Apriori classique

L'algorithme APriori est un algorithme d'exploration de données conçu en 1994, par Rakesh Agrawal et Ramakrishnan Sikrant, dans le domaine de l'apprentissage des règles d'association. Il sert à reconnaitre des propriétés qui reviennent fréquemment dans un ensemble de données et d'en déduire une catégorisation.

Apriori avec fragementation vertical

Vertical fragmentation is a technique used in distributed databases where a table is divided vertically into multiple tables based on the columns. This allows each table to be stored on a separate server, which can improve performance and scalability.

Algorithme close

L'algorithme CLOSE propose par Pasquier et al. [48, 49] est le premier algorithme permettant l'extraction des itemsets fermes frequents. Cet algorithme se base sur les generateurs minimaux pour calculer les itemsets fermes.
